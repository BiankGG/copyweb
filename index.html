Construye cosas increíbles

Somos GitHub, la empresa detrás del Registro npm y la CLI de npm. Se los ofrecemos a la comunidad de forma gratuita, pero nuestro trabajo diario es crear y vender herramientas útiles para desarrolladores como usted.

Lleva tu desarrollo de JavaScript a un nivel superior

Comience hoy de forma gratuita o actualice a npm Pro para disfrutar de una experiencia de desarrollo de JavaScript premium, con funciones como paquetes privados.

Ofrezca lo mejor del código abierto a usted, a su equipo y a su empresa.

Con la confianza de más de 17 millones de desarrolladores en todo el mundo, npm se compromete a hacer que el desarrollo de JavaScript sea elegante, productivo y seguro. El Registro npm gratuito se ha convertido en el centro para compartir ódigo JavaScript y, con más de dos millones de paquetes, el registro de software más grande del mundo. Nuestras otras herramientas y servicios llevan el Registro y el trabajo que usted realiza a su alrededor al siguiente nivel.
